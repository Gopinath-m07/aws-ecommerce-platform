# Smart Nenotech E-Commerce Platform

## 📌 Project Overview

This project involved designing and deploying a scalable e-commerce platform for Smart Nenotech (Client: Jude) using Amazon Web Services. The system was built to ensure high availability, scalability, secure data storage, and efficient application deployment in a cloud environment.

---

## 👨‍💻 Role

Cloud Engineer

## ⏱ Duration

3 Months

## 🏢 Client

Smart Nenotech (Client: Jude)

---

## 🛠️ Technologies Used

* Amazon EC2
* Amazon S3
* Amazon RDS
* AWS Elastic Beanstalk
* Amazon CloudFront
* Amazon CloudWatch
* IAM
* VPC (Basic Configuration)

---

## 🧠 Architecture

User → CloudFront → Elastic Beanstalk → EC2 → RDS
Static Files → Stored in S3
Monitoring → CloudWatch

---

## ⚙️ Implementation Steps

1. Launched EC2 instances for application hosting.
2. Configured Security Groups and IAM roles for secure access.
3. Created S3 buckets to store static files and application assets.
4. Set up Amazon RDS for relational database management.
5. Deployed the application using AWS Elastic Beanstalk.
6. Integrated CloudFront CDN to improve content delivery speed.
7. Configured CloudWatch for monitoring logs and performance metrics.
8. Tested scalability and availability of the application.

---

## 🚀 Key Features

* Scalable cloud infrastructure
* Secure storage and database management
* Auto deployment using Elastic Beanstalk
* Content delivery optimization using CloudFront
* Monitoring and logging using CloudWatch
* High availability architecture

---

## 🎯 Outcome

Successfully deployed a scalable and reliable e-commerce application on AWS cloud infrastructure with improved performance, high availability, and secure data management.
